# appclima-python — Proyecto educativo

> `appclima-python` es un proyecto **educativo, open source y sin fines de
> lucro** (licencia MIT). No tiene modelo de negocio ni monetización: su valor
> está en enseñar. Este documento describe su propósito y a quién sirve.
>
> **Última actualización**: 2026-07-02

## 1. Propósito

- Enseñar los **fundamentos de Python** de forma progresiva y práctica.
- Mostrar cómo un mismo problema (consultar el clima de una ciudad) se resuelve
  con distintos conceptos a lo largo de 7 etapas.
- **Alternativas actuales**: tutoriales dispersos que no siempre construyen un
  proyecto completo y coherente de principio a fin.

## 2. Público objetivo

| Público                     | Descripción                                                    |
| --------------------------- | ------------------------------------------------------------- |
| Estudiantes de programación | Personas aprendiendo Python desde cero, paso a paso.          |
| Principiantes autodidactas  | Quienes buscan un proyecto guiado y ejecutable para practicar. |
| Docentes / mentores         | Quienes necesitan un ejemplo incremental para enseñar.        |

## 3. Propuesta de valor

- Un proyecto real y ejecutable que crece concepto a concepto: de condicionales
  a Programación Orientada a Objetos.
- **Concepto de alto nivel**: "un curso de fundamentos de Python en forma de
  aplicación de clima".

## 4. Contenido (las 7 etapas)

| Etapa | Qué enseña                                       |
| ----- | ------------------------------------------------ |
| 1     | Control de flujo básico (condicionales).         |
| 2     | Ciclos e iteraciones (bucles).                   |
| 3     | Creación y uso de funciones.                     |
| 4     | Manejo de arreglos y persistencia (listas).      |
| 5     | Uso de diccionarios.                             |
| 6     | Consumo de APIs (OpenWeatherMap con `requests`). |
| 7     | Programación Orientada a Objetos (clase `Clima`).|

## 5. Distribución

- Repositorio público en GitHub:
  <https://github.com/brayandiazc/appclima-python>.
- Licencia MIT: cualquiera puede usar, copiar, modificar y compartir el código.

## 6. Modelo económico

- **Sin monetización**: proyecto gratuito y open source, sin planes de pago,
  sin publicidad y sin recolección de datos.
- **Costos**: mínimos; solo requiere una API KEY gratuita de OpenWeatherMap
  para las etapas 6 y 7.

## 7. Cómo se mide el éxito

- Personas que completan las 7 etapas y comprenden cada concepto.
- Reutilización del proyecto como material de enseñanza.
- Contribuciones y estrellas en el repositorio.
